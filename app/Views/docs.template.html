Routing

SigmaPHP provides a simple, magic-free router that allows you to define clean and expressive routes for your application. 

        
            $router->get('/users', [UserController::class, 'index']);
            $router->post('/users', [UserController::class, 'store']);
        
    
        
            $ ./bin/sigma-cli run
            [Fri Jan  2 10:15:26 2026] PHP 8.2.28 Development Server (http://localhost:8888) started

Routes are matched in the order they are defined, giving you full control over request handling.
